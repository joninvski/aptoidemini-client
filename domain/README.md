domain module
=============

The domain objects used by AptoideMini Client.

This library should not depend on anything non-standard and widely available

## Gradle targets

* Build the jar file

    ```
    ./gradlew
    ```

* Execute a codecheck (lint)

    ```
    ./gradlew :domain:check
    ```

* Execute a tests (TODO)

    ```
    ./gradlew :domain:test
    ```
